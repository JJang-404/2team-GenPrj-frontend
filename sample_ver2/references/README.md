# Reference Mapping

`sample_ver1`은 참고 이미지를 직접 복제하지 않고, 원본 `../샘플` 폴더를 taxonomy 관점에서 재해석한다.

- `premium`: 고급화 음식점/카페 포스터
- `general_food`: 일반 홍보형과 가격 강조형 포스터
- `vintage`: 레트로/클래식 포스터
- `delivery`: 일반 홍보 이미지 중 CTA/할인 구조를 가진 사례를 delivery로 추상화
- `menu_board`: 단일 메뉴, 가격표, 복수 카드형 메뉴판

각 하위 폴더 README에는 원본 위치와 추상화 포인트를 적어 두었다.
