  프론트엔드 프로젝트 루트 폴더(예: D:\01.project\2team-GenPrj-frontend_United\United1\react)로 이동하여 다음 명령어를
  실행해 주세요.

  방법 A: npm 사용 시
   1 npm install axios

  방법 B: yarn 사용 시
   1 yarn add axios

  3. 확인 사항
  설치 후 react/package.json 파일을 열어 dependencies 항목에 axios가 추가되었는지 확인해 보세요.

   1 "dependencies": {
   2   "axios": "^1.x.x",
   3   ...
   4 }

   1 rmdir /s /q node_modules
   2 npm install

  이렇게 하면 axios를 정상적으로 불러올 수 있게 되어 에러가 해결될 것입니다. 조
