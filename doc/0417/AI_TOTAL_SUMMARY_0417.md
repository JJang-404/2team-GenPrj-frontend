# 2026-04-17 AI 배경 생성 및 디자인 지능화 통합 보고서

본 문서는 금일 진행된 AI 배경 생성 엔진 고도화와 그에 따른 디자인 자동 최적화 시스템의 모든 기술적 내역을 통합하여 기술합니다.

##  1. AI 생성 파이프라인 마이그레이션 및 최적화

기존의 단순 텍스트 기반 생성을 탈피하여, 제품의 형태를 가이드로 사용하는 고퀄리티 생성 환경을 구축했습니다.

### **핵심 변경 사항**
- **ComfyUIJobs API 전환**: 긴 생성 시간을 안정적으로 처리하기 위해 비동기 폴링(`_pollJobAsync`) 로직을 `modelApi.js`에 구현했습니다.
- **노이즈 제거(Denoising) 강도 최적화**: 
  - 배경 생성 시 **1.0 (최대치)**을 적용하여 제품 가이드를 참조하되 배경은 100% 새롭고 화려하게 생성되도록 설정했습니다.
  - 정면 변환(Transformation) 시에는 제품 보호를 위해 **0.3**의 낮은 강도를 유지합니다.

##  2. 업종별 프롬프트 자동 최적화 시스템

사용자의 업종 정보를 분석하여 가장 어울리는 배경 프롬프트를 자동으로 조립합니다.

### **프롬프트 관리 모듈 (`prompts.ts`) [NEW]**
- **위치**: `react/src/modules/editing/constants/prompts.ts`
- **로직**: `industry` 데이터 내에 '커피, 카페, 빵, 디저트' 등의 키워드가 포함되면 **Dark Moody Cafe** 프롬프트가 적용되고, 그 외에는 **Bright Minimalist** 프롬프트가 적용됩니다.
- **장점**: 프롬프트 수정을 위해 메인 로직을 건드릴 필요 없이 상수 파일만 업데이트하면 됩니다.

##  3. 지능형 시각 대비 (Auto-Contrast) 시스템

배경 이미지의 밝기에 따라 텍스트 가독성을 실시간으로 확보하는 디자인 지능화를 구현했습니다.

### **밝기 분석 및 색상 전환 로직**
- **기술 스택**: `imageAnalysis.ts` 유틸리티를 통한 W3C 표준 휘도(Luminance) 계산.
- **자동 전환 정책**:
  - **AI 배경/단색**: 휘도 분석 결과 128 미만(어두움)일 경우 텍스트를 **흰색(#FFFFFF)**으로, 그 이상일 경우 **검정색(#000000)**으로 강제 전환합니다.
  - **그라데이션/다중색(Pastel)**: 분석 오류 방지를 위해 **분석을 생략하고 기본색(검정색)**을 유지하도록 예외 처리를 완료했습니다.
