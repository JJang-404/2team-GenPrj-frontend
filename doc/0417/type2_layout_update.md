# Type 2 레이아웃 스타일 변경 내역 (하단 헤더 전환)

- **작성 일자**: 2026-04-17
- **대상 레이아웃**: Type 2 (`SingleCompactLayout`)

## 1. 개요
Type 2 레이아웃의 컨셉을 '상단 헤더'에서 '하단 헤더' 스타일로 전환하였습니다. 가게 이름과 광고 문구를 제품 이미지 아래쪽으로 이동시켜 시각적인 계층 구조(Hierarchical Spec)를 강화하였습니다.

## 2. 세부 변경 사항

### 2.1 좌표 수정
가게 이름과 광고 문구가 캔버스 하단 영역에 위치하도록 기본 좌표를 수정하였습니다.
- **가게 이름 (store)**: 
    - 위치: `y: 10%` (상단) → `y: 80%` (하단 제품 아래)
    - 정렬: 좌측 정렬(`left`) 유지
    - 스타일: 기존의 **-3° 회전 효과** 유지
- **광고 문구 (slogan)**: 
    - 위치: `y: 21%` (상단) → `y: 92%` (최하단)
    - 정렬: 좌측(`left`) → **중앙 정렬(`center`)** 및 너비 100% 확장

### 2.2 라벨 및 명칭 업데이트
`WireframeChoiceCard`에서 사용자가 구도를 직관적으로 파악할 수 있도록 라벨을 업데이트하였습니다.
- **타이틀**: `Type 2 · 컴팩트 헤더` → `Type 2 · 하단 헤더`
- **설명**: `하단 헤더 블록 + 컴팩트 제품 배치`

## 3. 수정된 파일 목록
| 파일 경로 | 수정 내용 |
| :--- | :--- |
| `editorFlow.ts` | `LEGACY_TEXT_PLACEMENTS` 내 Type 2 좌표 및 속성 업데이트 |
| `wireframeTextPlacements.ts` | `WIREFRAME_TEXT_PLACEMENTS` 내 Type 2(index 1) 좌표 일치화 |
| `WireframeChoiceCard.tsx` | 레이아웃 선택 카드의 타이틀 라벨 업데이트 |
| `SingleCompactLayout.jsx` | `zonePositions` 부재 시를 대비한 하단 배치 fallback 로직 보강 |

## 4. 검증 결과
- 구도 선택 시 프리뷰 카드이미지와 메인 편집 화면(EditorCanvas)의 위치가 일치함을 확인.
- 제품 슬롯 영역(y: 8% ~ 76%)과 텍스트 영역이 겹치지 않고 정상적으로 배치됨.
