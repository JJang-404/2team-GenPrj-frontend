# AI 이미지 생성 프롬프트 파이프라인 구조 개선 (04/20)

## 개요
기존에 프론트엔드의 `prompts.ts`에 정의된 고퀄리티 배경 프롬프트(`SCENE_PROMPTS.UNIVERSAL`)가 너무 강력한 형태 제어 키워드를 지니고 있어, 사용자의 자유로운 배경 생성 요청(예: "자연 풍경")을 무시하고 스튜디오 톤으로 고정시키는 문제가 있었습니다. 이를 개선하기 위해 **LLM 동적 배경 묘사 기술**을 파이프라인에 적용하여, 사용자의 의도대로 유연하고 아름다운 뷰를 렌더링하면서도 최고 화질을 보장하도록 아키텍처를 재설계하였습니다.

## 주요 변경 사항

### 1. 프론트엔드 템플릿 다이어트 및 치환자 교체 (`prompts.ts`)
* **형태 통제 키워드 제거:** 완전히 추상적인 스튜디오 배경을 강제하던 `(unrecognizable abstract background:1.5)`, `(no distinct shapes:1.6)` 등의 강력한 형태 및 조명 통제 고정 키워드를 모조리 걷어냈습니다.
* **품질 보장 키워드만 보존:** `(masterpiece:1.3), (best quality:1.3), 8k resolution, flawless render` 과 같이 필수적으로 그림의 퀄리티를 폭발시키는 핵심 설정값만 뼈대(`UNIVERSAL` 템플릿)로 남겼습니다.
* **치환자 교체:** 단순 번역만 하던 `{GPT_TRANSLATED_KEYWORDS}` 치환자 대신, 프롬프트 동적 대체를 명시하는 `{LLM_EXPANDED_BACKGROUND}` 치환자를 새로 도입했습니다.

### 2. 백엔드 LLM 동적 배경 묘사 로직 추가 (`app/restapi/_model_comfyui.py`)
`_build_comfyui_prompt_bundle` 파이프라인 함수에 사용자 친화적이고 동적인 프롬프트 생성 능력을 부여했습니다.
* **배경 확장 분기 추가:** 전달받은 `positive_prompt` 템플릿 텍스트 내에서 `{LLM_EXPANDED_BACKGROUND}` 구문이 발견되면 LLM 동적 배경 모드를 시작합니다.
* **OpenAI를 활용한 배경 묘사:** 단순히 "자연 풍경" 이라는 짧은 한글 단어를 영어로 1:1 번역하는 데 그치지 않고, 백엔드 내부의 `OpenAiJob.build_background_prompt_bundle()` 메서드를 가동합니다. LLM(ChatGPT)이 사용자의 짧은 입력값("자연 풍경")을 토대로 맑은 호수, 햇살, 구름 등의 **스테이블 디퓨전용 고품질의 풍부한 배경 묘사 영문 텍스트 40여 단어로 상상하고 확장**해 줍니다.
* **최종 템플릿 조립:** 백엔드 레이어에서 LLM이 상상해낸 웅장한 배경 텍스트 더미를 프론트엔드가 보내준 품질 보장 뼈대 템플릿의 `{LLM_EXPANDED_BACKGROUND}`에 교체결합(Replace)하여 최종 ComfyUI 워크플로우에 꽂아 넣습니다.

## 기대 효과
* **완벽한 사용자 창의성 지원:** 사용자가 "꽃밭", "카페", "바닷가" 등 어떠한 배경을 원하든지, LLM이 그 테마에 맞춰서 가장 예쁜 디테일(조명, 구조물 등)을 알아서 채워 구사해 줍니다.
* **압도적인 생성 퀄리티 유지:** 풍부한 배경 묘사 뒤에 프론트엔드에서 고정해 둔 초고화질, 프로페셔널 렌즈 세팅 키워드가 항상 안전하고 견고하게 같이 묶여들어가므로 저화질이나 기괴한 이미지가 나오지 않습니다.
* **시스템 구조의 고도화:** LLM 요약 오작동 버그를 피하면서 동시에 ChatGPT의 창의성은 100% 활용하는 가장 이상적인 ComfyUI Prompt Injection 파이프라인이 완성되었습니다.
