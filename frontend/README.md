# AD-GEN PRO

AI 기반 광고 이미지 초안(Draft)을 실시간으로 생성·미리보기할 수 있는 웹 애플리케이션입니다.  
가게 이름, 제품 이미지, 슬로건, 브랜드 컬러를 입력하면 여러 가지 포스터 레이아웃을 즉시 확인할 수 있습니다.

## 주요 기능

| 기능 | 설명 |
|---|---|
| **실시간 포스터 미리보기** | 입력값이 바뀔 때마다 우측 Draft Preview에 즉시 반영 |
| **4가지 디자인 컨셉** | Premium / Retro / Modern / Vivid 스타일 중 선택 |
| **3가지 비율** | 1:1 · 4:5 · 9:16 인스타그램 피드/정방형/스토리 규격 지원 |
| **최대 6개 프레임 생성** | 슬라이더로 동시에 볼 초안 수 조절 (1~6개) |
| **제품 이미지 업로드** | 여러 제품 이미지를 업로드해 포스터에 자동 배치 |
| **배경 제거** | Python 서버(`rembg`)를 통해 이미지 배경 원클릭 제거 |
| **AI 슬로건 생성** | 가게 이름·제품명 기반으로 광고 문구 자동 생성 |
| **디자인 선택** | 마음에 드는 초안을 선택하면 상단에 고정 표시 |

### 백엔드(배경 제거 서버) 실행

배경 제거 기능을 사용하려면 Python 서버를 **별도 터미널**에서 실행해야 합니다.

```bash
# 1. 의존성 설치
pip install fastapi uvicorn rembg onnxruntime

# 2. 서버 실행 (프로젝트 루트에서)
python server.py
```

서버는 `http://localhost:8000` 에서 실행되며, 프론트엔드와 동시에 켜져 있어야 배경 제거 버튼이 작동합니다.

#### GPU 가속 설정 (선택 사항)

배경 제거 처리 속도를 높이려면 `onnxruntime-gpu`를 설치합니다.  
**NVIDIA GPU + CUDA** 환경이 필요합니다.

```bash
# 기존 CPU 버전 제거 후 GPU 버전 설치
pip uninstall onnxruntime
pip install onnxruntime-gpu
```

GPU 사용 가능 여부 확인:

```bash
python -c "import onnxruntime; print(onnxruntime.get_available_providers())"
# 출력 결과에 CUDAExecutionProvider 가 포함되면 GPU 사용 가능
```

> **참고:** `onnxruntime-gpu`가 설치되어 있더라도 `server.py`의 `remove()` 호출 시  
> provider를 명시하지 않으면 CPU로 동작합니다.  
> GPU를 명시적으로 활성화하려면 `rembg.new_session()`의 `providers` 파라미터를 설정해야 합니다.

## 주요 기능 상세

### 디자인 컨셉 선택

4가지 컨셉 중 하나를 선택합니다. 선택 즉시 모든 프리뷰에 반영됩니다.

| 컨셉 | 배경 스타일 | 적합한 용도 |
|---|---|---|
| **Premium** | 짙은 슬레이트 다크 | 고급 브랜드, 명품, 뷰티 |
| **Retro** | 아이보리 warm 톤 | 빵집, 카페, 레트로 감성 |
| **Modern** | 순백색 미니멀 | IT, 테크, 클린 브랜드 |
| **Vivid** | 레드-핑크 그라디언트 | 식음료, 이벤트, 프로모션 |

---

### 브랜드 컬러 및 프레임 수 설정

- **브랜드 컬러**: 컬러 피커로 색상 선택 → 포스터 내 가게 이름 텍스트에 즉시 적용
- **프레임 수**: 슬라이더로 1~6개 조절 → 각 초안마다 서로 다른 포스터 레이아웃이 적용됨

포스터 레이아웃은 4가지 스타일이 순환됩니다:
1. **정갈한 중앙 배치** — 이미지 중앙, 정보 하단
2. **대각선 타이틀** — 가게 이름을 크게 기울여 배치
3. **역동적 기울기** — 이미지에 회전 효과
4. **오버레이 중첩** — 이미지 위에 텍스트 오버레이

---

### 광고 정보 입력

| 입력 항목 | 설명 |
|---|---|
| **가게 이름** | 포스터에 브랜드 컬러로 표시됨 |
| **광고 슬로건** | 포스터 상단에 표시. 비워두면 `AI 생성` 버튼으로 자동 생성 가능 |
| **AI 상세 지시사항** | 추후 AI 이미지 생성 API 연동 시 사용할 추가 프롬프트 입력란 |

**AI 슬로건 생성 버튼:** 가게 이름과 첫 번째 제품명을 조합하여 광고 문구를 자동으로 만들어줍니다.

### 제품 리스트 관리

여러 제품을 추가하여 포스터 한 장에 함께 배치할 수 있습니다.

| 기능 | 설명 |
|---|---|
| **이미지 업로드** | 클릭하여 로컬 이미지 파일 선택 (jpg, png, webp 등) |
| **이미지 삭제** | 업로드된 이미지 우상단 X 버튼으로 삭제 |
| **제품명 / 가격 / 설명** | 각 항목 직접 입력 |
| **이름/가격/설명 ON·OFF** | 포스터에 각 항목을 표시할지 개별 토글 |
| **제품 추가** | `+ 추가` 버튼으로 제품 무제한 추가 |
| **제품 삭제** | 휴지통 아이콘으로 개별 삭제 (최소 1개 유지) |

**사이드바 확장 모드**: 우측 상단 `상세 확장` 버튼 클릭 시 사이드바가 900px로 넓어지며  
제품 카드가 2열 그리드로 전환되어 더 넓게 편집할 수 있습니다.

### 배경 제거

업로드한 제품 이미지의 배경을 AI로 자동 제거합니다.

**동작 과정:**
1. 제품 이미지 업로드
2. `배경 제거` 버튼 클릭
3. 이미지를 Python 서버(`http://localhost:8000/remove-bg`)로 전송
4. `rembg` (U2Net 모델)로 배경 제거 처리
5. 배경이 투명하게 제거된 PNG 이미지로 교체

> **주의:** Python 서버(`server.py`)가 실행 중이어야 합니다. 서버가 꺼져 있으면 경고 메시지가 표시됩니다.

**사용 모델:** `rembg` 라이브러리의 `u2net` — 딥러닝 기반 Salient Object Detection 모델

---

### 디자인 선택

각 초안 카드 하단의 `이 디자인 선택` 버튼을 클릭하면:

- 선택된 카드에 파란 테두리가 적용되고 `디자인편집` 버튼으로 전환됩니다.
- 프레임 수를 줄여도 선택된 초안은 항상 목록 최상단에 유지됩니다.
- 다시 클릭하면 선택이 해제됩니다.

## 환경 변수 및 설정

별도의 `.env` 파일은 필요하지 않습니다.  
백엔드 서버 URL은 [src/App.jsx](src/App.jsx) 파일 내에 직접 작성되어 있습니다.

```js
// src/App.jsx - 38번째 줄
const res = await fetch('http://localhost:8000/remove-bg', { method: 'POST', body: formData });
```

서버 포트나 호스트를 변경하려면 이 URL과 `server.py`의 `uvicorn.run()` 포트 설정을 함께 수정하세요.

## 알려진 제한 사항

- **AI 슬로건 생성**은 현재 목업(mock) 데이터로 동작합니다. 실제 OpenAI API 연동은 미구현 상태입니다.
- **배경 제거 서버**는 기본적으로 CPU로 동작합니다. GPU 가속이 필요한 경우 위의 [GPU 가속 설정](#gpu-가속-설정-선택-사항) 항목을 참고하세요.
- **Generate AI Drafts** 버튼은 현재 UI 데모 상태이며 실제 이미지 생성 API와 연동되어 있지 않습니다.
- 제품 이미지를 업로드하지 않은 경우 포스터 내 이미지 영역은 비어있거나 플레이스홀더로 표시됩니다.
