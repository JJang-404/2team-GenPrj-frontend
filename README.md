# 광고 이미지 에디터 — React UI Demo

> `demo_ad_editor.html` 단일 파일로 실행되는 광고 이미지 편집 UI 데모입니다.  
> 빌드·서버·설치 없이 브라우저에서 바로 열 수 있습니다.

---

## 실행 방법

```
demo_ad_editor.html 파일을 브라우저에서 열기
```

별도 설치나 서버 실행 없이 파일 더블클릭으로 즉시 동작합니다.  
(React 18, Babel은 CDN으로 자동 로드됩니다.)

---

## 주요 기능

### 배경 이미지
- JPG/PNG 음식 사진 업로드
- 이미지 크기 슬라이더 (20% ~ 100%) — 축소 시 나머지 영역을 단색으로 채움
- 나머지 배경 색상 선택 (color picker)

### 텍스트 요소 편집
총 6개의 기본 텍스트 요소가 제공됩니다.

| 요소 | 기본값 |
|---|---|
| 날짜 | March. 31 |
| 타이틀 | DELICIOUS |
| 식당 이름 | 코드잇 |
| 음식 이름 | 블루베리 팬케이크 |
| 가게 정보 | 카페 코드잇 / 전화 / 영업시간 |
| 광고 문구 | 신선한 재료로 만든 홈메이드 팬케이크 |

**조작 방법**
- **드래그** — 텍스트 요소를 마우스로 드래그해서 위치 이동
- **더블클릭** — 캔버스 위에서 텍스트 직접 편집 (Enter 확정 / Esc 취소)
- **눈 아이콘** — 요소 숨기기 / 보이기 토글

**스타일 편집 (왼쪽 패널)**
- 텍스트 정렬: 좌 / 중앙 / 우
- 텍스트 내용 수정
- 폰트 크기 슬라이더
- 글자 색상 picker
- 박스 배경 투명도 슬라이더
- X / Y 위치 직접 입력 (%)

### 하단 물결 설정
- 물결 표시 on/off 토글
- 물결 위치 (캔버스 상단 ↔ 하단)
- 물결 진폭 (0 = 완전 평평한 직선)
- 물결 어둠 (투명도)

### API 전송 (Mock)
- **이미지 생성하기** 버튼 클릭 시 FastAPI 백엔드로 전송될 JSON 페이로드를 미리 확인할 수 있습니다.
- 숨긴 요소는 전송에서 제외됩니다.

## 실제 서비스 연동 구조 (백엔드)

데모에서 "이미지 생성하기" 버튼은 아래와 같은 JSON을 FastAPI 백엔드로 전송합니다.

```json
POST /api/generate-ad
{
  "texts": {
    "title": "DELICIOUS",
    "restaurant": "코드잇",
    "food_name": "블루베리 팬케이크",
    ...
  },
  "positions": {
    "title":  { "x": "50.0%", "y": "7.0%" },
    "restaurant": { "x": "50.0%", "y": "21.0%" },
    ...
  },
  "styles": {
    "title": { "fontSize": 8.2, "color": "#ffffff" },
    ...
  },
  "wave": {
    "enabled": true,
    "amplitude": 28,
    "offsetY": 82,
    "opacity": 0.55
  },
  "image": "(업로드된 이미지 포함)"
}
```

백엔드(FastAPI + PIL)는 이 값을 받아 1080×1350px PNG를 생성하여 반환합니다.

## 파일 구성

```
demo_ad_editor.html     # 이 파일 하나로 전체 UI 동작
```

### 의존성 (CDN 자동 로드)
| 라이브러리 | 버전 | 용도 |
|---|---|---|
| React | 18 | UI 상태 관리 |
| ReactDOM | 18 | 렌더링 |
| Babel Standalone | - | JSX 트랜스파일 |

> **주의**: `font/` 폴더의 TTF 파일은 이 HTML에서 사용하지 않습니다.  
> TTF 폰트는 서버(PIL) 렌더링 전용입니다.
