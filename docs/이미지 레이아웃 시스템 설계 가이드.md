# 이미지 레이아웃 시스템 설계 가이드 (Content-Driven & Center-Anchored)

본 문서는 이미지의 원본 비율을 보존하며 와이어프레임 슬롯(Slot)을 콘텐츠에 맞춰 유동적으로 재조정하고, 고정된 기준점(Anchor Point)을 중심으로 자동 배치하기 위한 수학적 규칙을 정의합니다.

---

## 1. 핵심 설계 원칙

1. **콘텐츠 주도형 슬롯 (Content-driven Slot):** 슬롯의 크기는 고정되지 않으며, 이미지를 가이드라인 내에서 최대화(Cover)한 후 그 결과값에 맞춰 슬롯 크기를 최종 확정한다.
2. **중앙선 고정 정렬 (Center-line Anchoring):** 그룹 내 요소들이 만나는 '중앙선' 또는 '겹침 중심'을 와이어프레임의 고정 좌표($C_x$)에 일치시킨다.
3. **여백 없는 조립 (No-Gap Policy):** 이미지를 슬롯에 맞출 때 너비와 높이 중 비율이 더 큰 쪽을 선택하는 `max` 스케일링을 적용하여 빈 공간을 원천 차단한다.

---

## 2. 공통 스케일링 로직 (Scaling Engine)

이미지($i_w, i_h$)와 가이드라인 슬롯($s_w, s_h$)이 주어졌을 때의 배율 계산입니다.

### 2.1 확대/축소 배율 ($AR$) 결정
이미지가 슬롯을 빈틈없이 채우기 위한 최소 배율을 구합니다.
* **Type 4 (반반 조립):** $S_{target} = s_w \times 2$ (두 조각 합산 기준 너비)
* **그 외 타입:** $S_{target} = s_w$
$$AR = \max\left(\frac{S_{target}}{i_w}, \frac{s_h}{i_h}\right)$$

### 2.2 스케일링 된 이미지 크기
$$W_{scaled} = i_w \times AR$$
$$H_{scaled} = i_h \times AR$$

---

## 3. 타입별 상세 배치 수식

### 3.1 Type 4: 중앙선 기준 반반 조립 (Half-Crop)
이미지의 중앙선과 그룹의 중앙선을 일치시켜 조립합니다.

* **최종 슬롯 너비 ($w_{final}$):** $W_{scaled} / 2$
* **캔버스 좌표 (Destination):**
    * **왼쪽 조각 (1-1):** $dx = C_x - w_{final}, \quad dw = w_{final}$
    * **오른쪽 조각 (1-2):** $dx = C_x, \quad dw = w_{final}$
* **이미지 소스 (Source):**
    * **왼쪽 조각:** $sx = 0, \quad sw = i_w / 2$
    * **오른쪽 조각:** $sx = i_w / 2, \quad sw = i_w / 2$

### 3.2 Type 3: 중앙선 기준 겹침 배치 (Overlapping)
두 이미지의 중첩 영역 중심을 그룹의 중앙선에 일치시킵니다.

* **겹침 너비 ($O_w$):** $(w_{1\_1} + w_{1\_2}) \times 0.2$
* **캔버스 좌표 (Destination):**
    * **왼쪽 요소 (1-1):** $dx = C_x + \frac{O_w}{2} - w_{1\_1}$
    * **오른쪽 요소 (1-2):** $dx = C_x - \frac{O_w}{2}$
* **특이사항:** 슬롯이 이미지 크기에 따라 늘어나더라도 겹침의 중심은 $C_x$를 유지한다. 또한 항상 우측 슬롯의 이미지가 앞으로 나오도록 한다.

---

## 4. 표준 배치 프로세스 (6-Step Workflow)

1. **중심점 확정:** 와이어프레임에서 각 요소/그룹의 중심점($C_x, C_y$) 비율을 도출한다.
2. **초기 슬롯 지정:** 슬롯의 초기 가이드라인 크기($s_w, s_h$)를 설정한다.
3. **이미지 초기화:** 이미지 로드 후 원본 크기($i_w, i_h$)를 파악한다.
4. **최종 크기 도출:** 레이아웃 타입에 따른 $AR$과 $W_{scaled}$를 계산한다.
5. **슬롯 동기화:** 계산된 이미지 크기에 맞춰 슬롯의 최종 $w, h$를 업데이트한다.
6. **좌표 역산 및 배치:** 고정 중심점($C_x$)과 타입별 오프셋 수식을 적용해 최종 $x, y$를 결정한다.
